# beam-stream

Media streaming and management API for Beam. Acts as JSON API for Beam frontends to interact with Beam's ,media-related backend services.

## Development

### Prerequisites

- Rust and Cargo

### Setup

```sh
cargo run
```

or

```sh
cargo watch -x run
```
